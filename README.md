# CODENAME-ASUKA - A PARRY-BASED BULLET HELL
Danmaku (bullet hell) inspired by ULTRAKILL, Undertale, and everything Fromsoft.

# FEATURES (COMPLETED):
- Deflective Parrying Mechanic
- 3 Fully Unique Attack types, built to compliment each other
- Snappy, Fast Combat, centered around parrying and precise timing.
- Style Meter
- Multi-phase Boss with souls-like attack patterns.

# PRE-REQUISITES:
- python 3.14
- pygame-ce (Installation: "pip install pygame-ce")

# HOW TO RUN:
- Run the game.py file.
